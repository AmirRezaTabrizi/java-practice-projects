# MaxInArray – Find the Largest Number

This Java program lets the user enter 5 numbers, stores them in an array, and finds the largest one using a **foreach loop**.

## Concepts
- Arrays  
- Foreach loop  
- `Arrays.toString()`  

## Example
=== Day 6 – Find Max in Array ===
Enter number 1: 12
Enter number 2: 45
Enter number 3: 8
Enter number 4: 67
Enter number 5: 23
Numbers: [12, 45, 8, 67, 23]
Maximum number: 67
